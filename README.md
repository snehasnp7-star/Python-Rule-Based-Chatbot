# Python Rule-Based Chatbot

## A command-line chatbot that responds to predefined user queries using Python dictionaries.

## Description

Python Rule-Based Chatbot is a simple console application developed to demonstrate the basics of chatbot development using Python. The chatbot matches user input with predefined responses stored in a dictionary and returns the appropriate reply.

This project is intended for beginners who want to understand dictionary operations, user input handling, loops, and conditional statements while building an interactive application.

---

## Features

- Interactive command-line chatbot
- Predefined responses stored in a Python dictionary
- Case-insensitive user input
- Handles common greetings and questions
- Returns a message for unsupported queries
- Beginner-friendly implementation

## How It Works

1. The chatbot stores predefined questions and responses in a Python dictionary.
2. The user enters a message through the console.
3. The input is converted to lowercase for case-insensitive matching.
4. If the message exists in the dictionary, the chatbot returns the corresponding response.
5. If the input is not recognized, an error message is displayed.
